Jeu Héros

Python Terminal Boucles Fonctions Aléatoire

Description

Jeu de combat en tour par tour entièrement développé en Python, jouable directement dans le terminal. Le joueur incarne un héros qui affronte un ennemi disposant de 50 points de vie. Chaque tour, le joueur peut attaquer (dégâts aléatoires), utiliser une potion (nombre limité) ou tenter de fuir. Le projet met en pratique les fondamentaux du langage Python : boucles, fonctions, conditions et gestion de l'aléatoire.

Fonctionnalités

⚔️
Combat en tour par tour
Système d'attaque avec dégâts aléatoires pour le héros et l'ennemi.
🧪
Système de potions
3 potions de soin disponibles, à utiliser stratégiquement.
🎲
Dégâts aléatoires
Utilisation du module random pour varier les combats.
🏆
Conditions de victoire / défaite
Détection automatique de la fin de partie.

Stack technique

Python 3 random (stdlib) Terminal I/O

Extrait de code

import random

def attaquer(attaquant, defenseur):
    degats = random.randint(5, 15)
    defenseur["pv"] -= degats
    print(f"{attaquant['nom']} inflige {degats} dégâts !")

def utiliser_potion(joueur):
    if joueur["potions"] > 0:
        joueur["pv"] = min(joueur["pv"] + 20, 100)
        joueur["potions"] -= 1
    else:
        print("Plus de potions disponibles !")

Ce projet m'a permis de consolider mes bases en Python : structuration du code en fonctions, utilisation de dictionnaires pour modéliser les entités du jeu, et gestion du flux du programme avec des boucles while.

MexiGo

HTML5 CSS3 JavaScript Responsive Site vitrine Font Awesome

Description

MexiGo est un site vitrine multi-pages pour une agence de voyage fictive spécialisée dans les séjours au Mexique. Il comporte plusieurs sections : présentation de l'agence, équipe de conseillers, services proposés, tarifs, témoignages clients et formulaire de contact complet. L'objectif était de construire un site professionnel et cohérent en HTML / CSS pur, sans framework, avec une navigation fluide entre les pages.

Pages & Sections

🏠
Accueil
Landing page avec hero, présentation, équipe, services, tarifs et témoignages.
🏄
Activités
Page dédiée aux excursions et activités disponibles au Mexique.
🍽️
Gastronomie
Page présentant la cuisine mexicaine et les spécialités locales.
✉️
Formulaire de contact
Champs nom, email, destination, message — avec validation HTML5 native.
👥
Équipe conseillers
Présentation des 3 conseillers avec photo, spécialité et zone géographique.
💶
Grille de tarifs
3 packs (Conseil 59€, Essentiel 149€, Sur-Mesure) avec descriptions.

Stack technique
